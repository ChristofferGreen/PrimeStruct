Vector/map de-builtinization roadmap note: the target state is that, aside from minimal runtime/memory substrate, C++ compiler/backend code does not know collection semantics for `vector` or `map`; that behavior should live in stdlib `.prime` files.
SoA de-builtinization roadmap note: the target state for `soa_vector` is also a stdlib `.prime` implementation.
